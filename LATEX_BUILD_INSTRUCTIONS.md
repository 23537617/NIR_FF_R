## Способы компиляции

### Способ 1: Онлайн-компиляторы (рекомендуется)

#### Overleaf (https://www.overleaf.com/)
1. Зарегистрируйтесь на overleaf.com (бесплатный аккаунт)
2. Создайте новый проект "Upload Project"
3. Загрузите файл `crypto_research_diploma.tex`
4. Загрузите папку `crypto_charts/` в корень проекта
5. Нажмите "Recompile" - документ будет скомпилирован автоматически

**Преимущества:**
- Не требует установки LaTeX
- Все пакеты уже установлены
- Удобное редактирование в браузере
- Автоматическое сохранение

#### ShareLaTeX, Papeeria
Аналогично Overleaf - загрузите файлы и компилируйте онлайн.

### Способ 2: Локальная установка LaTeX

#### Установка на Ubuntu/Debian
```bash
sudo apt update
sudo apt install texlive-full texlive-lang-russian texlive-science
```

#### Установка на macOS
```bash
# Установите MacTeX https://tug.org/mactex/
brew install --cask mactex
```

#### Установка на Windows
1. Скачайте MiKTeX с https://miktex.org/download
2. Установите, выбрав опцию автоматической установки пакетов
3. Добавьте русский языковой пакет через MiKTeX Console

#### Компиляция документа
```bash
cd /workspace
pdflatex crypto_research_diploma.tex
# При необходимости повторите для корректного отображения ссылок и оглавления
pdflatex crypto_research_diploma.tex
pdflatex crypto_research_diploma.tex
```

Результат: `crypto_research_diploma.pdf`

### Способ 3: Docker (для изолированной среды)

```bash
# Использование образа с TeX Live
docker run --rm -v $(pwd):/workspace blang/latex:ubuntu \
    pdflatex -interaction=nonstopmode /workspace/crypto_research_diploma.tex

# Повторите 2-3 раза для корректных ссылок
docker run --rm -v $(pwd):/workspace blang/latex:ubuntu \
    pdflatex -interaction=nonstopmode /workspace/crypto_research_diploma.tex
```

5. **Глава 2: Архитектура системы** - компоненты, схема потока данных
6. **Глава 3: Криптографические основы** - PKI, ECDSA, SHA-256
7. **Глава 4: Методы шифрования** - TLS, шифрование данных, RBAC
8. **Глава 5: Анализ безопасности** - угрозы, меры защиты
9. **Глава 6: Визуализация данных** - 8 графиков с анализом
10. **Глава 7: Сравнительная характеристика** - таблицы алгоритмов
11. **Глава 8: Рекомендации** - краткосрочные и долгосрочные улучшения
12. **Глава 9: Заключение** - выводы, научная новизна
13. **Список литературы** - 21 источник
14. **Приложения**:
    - Приложение A: Пример кода шифрования
    - Приложение B: Чеклист безопасности
    - Приложение C: Список графиков

## Используемые пакеты LaTeX

Документ использует следующие пакеты:
- `extreport` - класс документа с поддержкой 14pt шрифта
- `babel` с опцией `russian` - русская локализация
- `graphicx` - вставка изображений
- `float` - позиционирование рисунков
- `booktabs`, `tabularx` - оформление таблиц
- `listings` - листинги кода
- `hyperref` - гиперссылки
- `amsmath`, `amssymb` - математические формулы
- `geometry` - настройка полей
- `titlesec` - форматирование заголовков
- `setspace` - межстрочный интервал

В документе настроены следующие параметры:

- Формат страницы: A4

## Возможные проблемы и решения

### Ошибка: Package inputenc Error: Unicode character ...
**Решение:** Убедитесь, что файл сохранён в кодировке UTF-8.

### Ошибка: File 'crypto_charts/...' not found
**Решение:** Проверьте, что папка `crypto_charts/` находится в той же директории, что и `.tex` файл.

### Ошибка: Undefined control sequence
**Решение:** Установите недостающие пакеты через менеджер пакетов TeX.

### Неправильное отображение русских символов
**Решение:** Убедитесь, что установлен пакет `texlive-lang-russian` и используется компилятор с поддержкой UTF-8.

## Рекомендации по оформлению в дипломной работе

1. **Нумерация страниц:** В текущей версии нумерация включена автоматически
2. **Перекрёстные ссылки:** Используйте `\ref{}` для ссылок на рисунки и таблицы
3. **Цитирование:** В списке литературы 21 источник, отформатированный по ГОСТ
4. **Графики:** Все 8 графиков готовы для вставки, имеют подписи и источники
